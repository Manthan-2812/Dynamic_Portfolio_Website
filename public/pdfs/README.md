# PDFs Folder

Place your PDF files in this folder.

## Usage:
1. Add PDF files to this folder (e.g., report1.pdf, document.pdf)
2. In the admin panel, add PDF with path: /pdfs/your-filename.pdf
3. The preview and download will work automatically

## Example Paths:
- /pdfs/report1.pdf
- /pdfs/document.pdf
- /pdfs/financial-analysis.pdf

## Important:
- Do NOT use file:/// paths
- Files must be in this folder
- Paths must start with /pdfs/ and end with .pdf
