# Photography Folder

Place your photography images in this folder.

## Usage:
1. Add image files to this folder (e.g., photo1.jpg, landscape.png)
2. In the admin panel, add image with path: /photography/your-filename.jpg
3. The image will display in the gallery

## Example Paths:
- /photography/photo1.jpg
- /photography/landscape.png
- /photography/portrait.jpg

## Supported Formats:
- JPG, JPEG, PNG, GIF, WebP

## Important:
- Images must be in this folder
- Paths must start with /photography/
- Use relative paths, not file:///
