# Paintings Folder

Place your painting images in this folder.

## Usage:
1. Add image files to this folder (e.g., painting1.jpg, artwork.png)
2. In the admin panel, add image with path: /paintings/your-filename.jpg
3. The image will display in the gallery

## Example Paths:
- /paintings/painting1.jpg
- /paintings/artwork.png
- /paintings/landscape.jpg

## Supported Formats:
- JPG, JPEG, PNG, GIF, WebP

## Important:
- Images must be in this folder
- Paths must start with /paintings/
- Use relative paths, not file:///
